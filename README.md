# Cake Orders

Страница оформления заказа на торты. Оплата на сайте не предусмотрена: заказ будет записываться в Google Sheets и дублироваться уведомлением в Telegram. Проект ориентирован на деплой в Cloudflare Pages + Pages Functions.

Сейчас интерфейс работает как 2-экранный лендинг: сначала показывается Hero-блок, затем по клику открывается карточка заказа.

## Технологии
- Next.js (App Router)
- React + TypeScript
- Cloudflare Pages + Pages Functions

## Локальный запуск
1. `nvm use`
2. `npm install`
3. `npm run dev`

> Рекомендуемая версия Node.js: 20+ (минимум 18.18)

## Переменные окружения
Скопируй `.env.example` в `.env.local` и заполни значения:
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`
- `GOOGLE_SHEETS_ID`
- `GOOGLE_SERVICE_ACCOUNT_EMAIL`
- `GOOGLE_PRIVATE_KEY`

## Как будет работать отправка заказа
UI формы → `POST /api/order` → запись в Google Sheets + уведомление в Telegram → экран успеха.
